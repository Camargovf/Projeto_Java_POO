**/ Atenção *//



Esses exercicios ferem o SOLID.
Isso porque várias GODCLASS foram criadas, sendo assim o Principio
da Responsabilidade Unica é ferido,
alem de outras transgressoes.

Estando ciente da transgressa por motivos didaticos de POO.
De maneira alguma POO para uso real poderia ferir o SOLID por conta
 de engessamento de codigo e mudanca em reutilizacao
de code, fragil para manutencao.
